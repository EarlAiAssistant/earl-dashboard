# AI Logo Generation Prompt Research

## Best Practices for Logo Prompts

### 1. Structure Your Prompt

**Formula:**
```
[Style] + [Subject/Symbol] + [Industry context] + [Colors] + [Technical specs]
```

### 2. Key Elements to Include

| Element | Why It Matters | Example |
|---------|---------------|---------|
| **Style** | Sets visual direction | "minimal", "geometric", "modern" |
| **Symbol/Mark** | Core visual element | "abstract V shape", "sound wave" |
| **Industry** | Context for AI | "SaaS", "AI company", "tech startup" |
| **Colors** | Brand alignment | "teal and gold", "gradient" |
| **Technical** | Output quality | "vector", "scalable", "white background" |

### 3. Styles That Work for B2B SaaS

Based on successful logos in the space:

| Style | Examples | Vibe |
|-------|----------|------|
| **Geometric minimal** | Stripe, Linear, Notion | Clean, modern |
| **Abstract mark** | Slack, Figma | Memorable, unique |
| **Lettermark** | IBM, HBO | Professional |
| **Wordmark + icon** | Spotify, Dropbox | Versatile |

## Tips for Best Results

### Do:
- Use `--v 6` for latest Midjourney version
- Add `--style raw` for cleaner, less artistic output
- Specify "white background" or "transparent background"
- Mention "vector style" or "scalable"
- Reference similar brands for style direction
- Generate multiple variations, pick best elements

### Don't:
- Include too many concepts in one prompt
- Use complex metaphors AI won't understand
- Forget to specify colors
- Ask for text AND icon in same generation (do separately)

### Post-Generation:
1. Generate 10-20 variations
2. Pick top 3-5 concepts
3. Recreate in Figma/Illustrator as proper vectors
4. Test at various sizes (favicon to billboard)
