## Implementation Notes

### Conditional Logic

Skip emails if:
- User has high engagement (3+ generations)
- User already converted to paid
- User explicitly asked to stop emails

Upgrade to "power user" track if:
- 5+ generations in first week
- (Send testimonial request instead of conversion emails)

### Subject Line A/B Tests

Test these variations:
- Day 1: "Did you try it yet?" vs "Your first 60 seconds"
- Day 5: Quote vs "How Sarah publishes 4x more case studies"
- Day 10: "4 days left" vs "The math on your content"

### Personalization

Include when possible:
- Name
- Number of generations completed
- Days remaining in trial
- Content types generated

### Timing

- Send all emails at 9am in user's timezone
- If timezone unknown, default to 9am ET
